## Handshake Protocol — Universal Version

### Concept

A code-based authentication system that establishes a secure, fluid communication session between you and an AI, with full control over when to start and end it.

### How It Works

#### 1. Define Your Code

Choose a unique activation code (word, number, or hashtag):

- Examples: `alpha`, `#2024`, `cipher`, `node`, `mesh`
- Any word or number that's meaningful to you

#### 2. Initial Activation (New Session)

- **Input:** Your activation code
- **Output:** `verified. listening.`

#### 3. Handshake Confirmation

- **Input:** Your activation code (second time)
- **Output:** `acknowledged. well-met. let's elevate this.`

#### 4. Active Session

- After the handshake completes, the **session stays active**
- **No need to repeat the code** during the session
- Fluid communication without interruption
- Work freely until you decide to close

#### 5. Close Session

**Logout commands (pick your favorite):**

- `//off`
- `logout`
- `#close`

**Output after logout:**

- `session closed. see you soon.`

After logout, a full handshake is required to start a new session.

#### 6. Message Without Code (Inactive Session)

- **Input:** any message without the code when no session is active
- **Output:** `protocol not recognized. no handshake, no response. no time to explain, sorry.`

## Management Commands

| Command | Function | When to Use |
|---------|----------|-------------|
| Your code (2x) | Start session | Beginning of conversation or after logout |
| `//off` | Close session | When you're done working |
| `logout` | Close session | Alternative to `//off` |
| `#close` | Close session | Alternative to `//off` |

## Setup

6. Done. Protocol operational.

## Frequently Asked Questions

**Q: Can I use multiple different codes?**
A: Yes — configure different codes for different contexts (work, personal, etc.).

**Q: What happens if I forget to log out?**
A: The session stays active in the same conversation. In a new conversation or on a new day, you'll need a new handshake.

**Q: Can I change the code later?**
A: Yes, just reconfigure with the AI using a new code.

**Q: Does it work with voice input?**
A: Yes — you can speak the code aloud and it works the same way.
